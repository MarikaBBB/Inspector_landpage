Features

Checks Management

Inspector allows you to execute and manage SQL queries within your database or across multiple databases, with comprehensive support for any SqlAlchemy-compatible database. You can manage and view all defined checks within the application, including details such as names, statuses, and last run times. Additionally, Inspector enables you to define new checks with customizable SQL queries, set conditions for triggers and alerts, and specify the databases for query execution, ensuring a robust and flexible data validation process.

Check History

Review the logs and historical data of check executions to monitor and analyze the performance over time across multiple environments. This includes tracking success rates, run times, and error messages to ensure thorough testing and reliability of data.

Tagging

Organize checks with tags, allowing you to group and execute multiple checks simultaneously for efficient testing and monitoring.

Table List

Inspect the list of database tables, their records, and quality metrics, and perform operations such as profiling and validation.

Data Profiling and Reporting

Inspector streamlines the data profiling process by rapidly profiling database tables using ydata-profiling, enhancing the structure and integrity of your data. It tracks the history of data profiling activities, providing valuable insights into data characteristics and their evolution over time. Furthermore, Inspector utilizes the Pandas Profiling library to generate detailed reports on data profiling, complete with statistical summaries and visualizations, aiding in thorough data analysis and decision-making.
